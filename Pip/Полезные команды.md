# Полезные команды

Обновить pip:
    ```
    python -m pip install --upgrade pip
    ```

Показать зависимости:
    ```
    pip freeze
    ```

Сгенерировать requirements.txt:
    ```
    pip freeze > requirements.txt
    ```

Установка пакетов из requirements.txt:
    ```
    pip install -r requirements.txt
    ```
